MODO DE LUCHA

SE COMPARAN VELOCIDADES PARA VER QUIEN ATACA PRIMERO 

SE UTILIZAN ATAQUES DE MANERA ALEATORIA 

SE PUEDE LLEGAR A EVITAR EL ATAQUE SEGUN VELOCIDAD ( VELOCIDAD 20 = 20% DE ESQUIVAR)

EN CASO DE ATACAR Y ESQUIVAR, SEGUN EL ATAQUE SI TIENE EFECTO (CURACION) SE APLICA IGUAL.

EN CADA TURNO SE MUESTRA SI ACTUA EL JUGADOR O EL ENEMIGO

EN CADA TURNO SE MUESTRA EL NOMBRE DEL ATAQUE QUE SE HA USADO

SI EL ATAQUE HACE DAÑO SE MUESTRA A QUIEN GOLPEA Y CUANTO DAÑO HACE

SI EL ATAQUE ESQUIVA SE MUESTRA QUIEN HA ESQUIVADO

SI EL ATAQUE CURA SE MUESTRA LA VIDA RECUPERADA

DESPUES DE CADA RONDA SE MUESTRA EL ESTADO DEL COMBATE CON BARRAS DE VIDA

VIDA 0 = ACABAR COMBATE 

VIDA RIVAL < 0 GANAMOS
VIDA PROPIA < 0 PERDEMOS
